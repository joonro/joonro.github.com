Computing¶

I authored a couple of software packages for estimation of structural models, and I contributed to free and open-source scientific software projects such as Cython, IPython, Mayavi, and Pandas.

I am passionate about Free software and Reproducible Science. I am highly proficient in programming languages such as Python and R, other computer tools such as GNU/Linux, Git, and LaTeX.

Packages Authored¶

I have authored the following packages:

  • BLP-Python: a Python with Cython implementation of random coefficients logit model of Berry, Levinsohn and Pakes (1995).
  • Fast Cubic Spline Python: an implementation of fast spline interpolation algorithm of Habermann and Kindermann (2007) in Python with Cython.
