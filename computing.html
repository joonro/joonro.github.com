Computing¶

Author:Joon H. Ro

I am highly proficient in programming languages such as Python and R, and familiar with other computing tools such as GNU tools, Git, Emacs, LaTeX, and PowerShell.

I authored a couple of software packages for research and teaching, and I contributed to free and open-source scientific software projects such as Pandas, Cython, IPython, and Mayavi.

Software Packages Authored¶

I have authored the following packages:

BLP-Python
a Python with Cython implementation of random coefficients logit model of Berry, Levinsohn and Pakes (1995).
Fast Cubic Spline Python
an implementation of fast spline interpolation algorithm of Habermann and Kindermann (2007) in Python with Cython.
Org-Coursepack
Org-Coursepack provides a template for developing and managing teaching materials using Org mode, a major mode in GNU Emacs.
